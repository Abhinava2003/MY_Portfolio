Exploratory Data Analysis on Titanic Dataset

This project is an in-depth exploratory data analysis (EDA) of the Titanic dataset, aimed at uncovering survival trends and key feature correlations. The analysis was conducted using Python, leveraging Pandas for data manipulation, Matplotlib and Seaborn for advanced visualizations, and NumPy for numerical operations.

The dataset consisted of over 1,300 passenger records, and the project focused on cleaning missing values, handling categorical variables, and generating insights into survival probabilities based on factors like age, gender, passenger class, and family size. The analysis revealed significant correlations and patterns that influenced survival rates. The project also includes visually appealing plots such as histograms, heatmaps, and bar charts for better interpretation of the findings.

Python Pandas Matplotlib Seaborn Numpy
